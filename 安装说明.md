# Skills Manager 安装说明

## 安装方法

1. 双击 `Skills Manager-x.x.x-arm64.dmg` 打开
2. 把 **Skills Manager** 拖到 **Applications（应用程序）** 文件夹
3. 从启动台或应用程序文件夹打开

---

## 如果提示"已损坏，无法打开"

macOS Gatekeeper 对未经 App Store 分发的应用有限制，执行以下任意一种方法解决：

### 方法一：终端一行命令（推荐）

```bash
xattr -cr "/Applications/Skills Manager.app"
```

执行后直接双击打开即可。

### 方法二：系统设置手动允许

1. 打开 **系统设置 → 隐私与安全性**
2. 滚动到底部，找到"仍要打开"按钮
3. 点击后输入密码确认

### 方法三：右键打开

右键点击应用图标 → **打开** → 在弹窗中点击**打开**

---

## 系统要求

- macOS 10.12 以
- Apple Silicon (M 系列芯片) 或 Intel Mac
